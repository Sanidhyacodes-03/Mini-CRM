# 🚀 Mini CRM - Customer Relationship Management System

A modern **Full Stack Customer Relationship Management (CRM)** web application built using the **MERN Stack**. This application enables users to securely manage customer leads, monitor lead progress, and analyze business data through a clean and responsive interface.

---

### 🚀 Frontend
**https://mini-crm-liart-nu.vercel.app**

### ⚙️ Backend API
**https://mini-crm-api-hmtv.onrender.com**

### 💻 GitHub Repository
**https://github.com/Sanidhyacodes-03/Mini-CRM**

---

## 📖 Overview

Mini CRM is designed to simplify customer relationship management by providing an easy-to-use interface for managing leads. The application includes secure authentication, lead tracking, analytics, and complete CRUD functionality.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Sanidhyacodes-03/Mini-CRM.git
```

---

## Install Frontend

```bash
cd client
npm install
npm run dev
```

---

## Install Backend

```bash
cd server
npm install
npm start
```

---

# 🔑 Environment Variables

## Backend (.env)

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CORS_ORIGIN=http://localhost:5173
```

---

## Frontend (.env)

```env
VITE_API_URL=http://localhost:5000/api
```

# 📡 API Endpoints

## Authentication

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | `/api/auth/register` | Register a new user |
| POST | `/api/auth/login` | Login user |

---

## Leads

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/leads` | Get all leads |
| POST | `/api/leads` | Create a lead |
| PUT | `/api/leads/:id` | Update lead |
| DELETE | `/api/leads/:id` | Delete lead |

---

## Analytics

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/analytics` | Get dashboard analytics |
